# AkamaiOps - IaC / Automation

This repository contains code-based automation for Akamai operational configuration management.

## Scope
- Property inventory collection
- Version and activation status check
- Operational configuration validation
- Report export for review and evidence

## Purpose
- Reduce manual Akamai console operations
- Standardize operational checks
- Support code-based configuration management
- Improve repeatability and visibility

## Components
- PAPI-based Python automation scripts
- Environment-specific configuration files
- Execution outputs and evidence documents

## Expected Outcome
- Automated retrieval of Akamai resource status
- Consistent operational validation
- Reusable code for future Terraform-based expansion
